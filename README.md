# Facility Location Optimization Using PuLP

This project applies Integer Linear Programming (ILP) to solve a facility location optimization problem for a retail distribution network in Sri Lanka.

## Project Overview

The model determines:

- Which warehouses should be opened
- Which stores should be assigned to each warehouse
- How to minimize total logistics cost

The project was developed using Python and PuLP.

## Tools and Libraries

- Python
- PuLP
- Pandas
- Geopy
- Folium

## Features

- Facility location optimization
- Transportation cost minimization
- Capacity constraints
- Geographic visualization using Folium
- Network optimization modeling

## Results

- Optimal warehouses selected: Colombo and Kurunegala
- Total optimized logistics cost: LKR 10.44 million
